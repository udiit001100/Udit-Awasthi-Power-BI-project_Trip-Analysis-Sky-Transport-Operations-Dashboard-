# Udit-Awasthi-Power-BI-project_Trip-Analysis-Sky-Transport-Operations-Dashboard-
Design and develop an interactive Power BI dashboard that provides a high-level operational and financial overview of Sky Transport’s shipment and trip performance.

Dashboard Structure
Page 1: Financial insights
KPI Cards (Revenue, Cost, Profit, Miles)
Revenue by State
Profit by State
Interactive slicers (Date, State, Trip Type, Customer)
Page 2: Operational Insights
Trip Type Frequency
Longest Duration Trips
Empty Miles by State
Capacity Utilization Gauge
Page 3: Strategic Insights
Revenue by Customer
Most Profitable Routes
Revenue vs Cost Scatter Analysis
Bubble size represents profitability
Route-level breakdown for strategic planning

Key Findings & Insights
Based on analysis:

Certain states contribute significantly higher revenue but also carry higher operational costs.

Some routes demonstrate high revenue with controlled cost — these are high-profit corridors.

Empty miles represent a measurable inefficiency and opportunity for cost reduction.

Capacity utilization indicates potential to improve load optimization.

A small group of customers contributes a large share of total revenue.

Things we can make decisions on - Improvisations - 

Optimise high-cost routes with low profitability.

Reduce empty miles through better route planning and return-load optimisation.

Improve vehicle capacity utilisation above 85%.

Strengthen relationships with high-revenue customers.

Expand operations in consistently profitable regions.
